# mybooklist

## Project Setup

```sh
composer install
```

```sh
php artisan serve
```
### Make the migrations(and creating the DB if its the first time that you use the app)

```sh
php artisan migrate
```

### Refresh migrations

```sh
php artisan migrate:refresh
```
### Seeding the database

```sh
php artisan db:seed
```
### Refresh migrations and seed database

```sh
composer test
```
